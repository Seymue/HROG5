Справка
Шпаргалка по управлению и горячие клавиши
Ошибки
  • EXT — Проблема с внешней опорой 5 MHz
  • PLL — PLL не залочена на reference (нет “захвата”)
  • PARAM — Отправлена команда с некорректным параметром (диапазон/формат/единицы)
  • INT — Проблема с внутренним осциллятором
  • TUNE — Напряжение подстройки вышло за норму
  • CMD — Некорректная/неизвестная команда
  • OSC — Уровень мощности внутреннего генератора. Диапазон от +11,5 до +12,5 дБм.
  • LOCK — Напряжение детектора блокировки. Диапазон: от 0,2 до 0,35 В.
  • REF — Уровень мощности внешнего опорного сигнала. Диапазон от +10 до +16 дБм.
  • TUNE — Настройка напряжения порта на генераторе. Допустимый диапазон: +/- 5 В.
Быстрые команды
  • Синхронизация — SYNC
  • Сброс фазы — *RPHS
  • Очистка статуса — Сбрасывает ошибку, если устранена
Параметры
  • Частота — SET_FREQ / STEP_FREQ
  • Фаза — SET_PHASE / STEP_PHASE
  • Временной сдвиг — SET_TIME_OFFSET / STEP_TIME_OFFSET
Примечание
UI работает строго “под бек”: все данные статуса берутся через команду GET_STATUS. Если устройство не выбрано — команды заблокированы.
